# Real-World Manipulation Examples

## Family Court Manipulation

| Phase | Tactic |
|-------|--------|
| 1. False Emergency | Restraining order for "immediate danger" |
| 2. Communication Bombardment | Daily legal filings and court dates |
| 3. Solution Rejection | Mediation refused, coparenting dismissed |
| 4. Control Demands | Total custody or supervised visits only |
| 5. Punishment Escalation | Criminal charges, contempt proceedings |

**Domain Coefficient: 2.5x** (highest manipulation intensity)
